## Key Terms Explained

### Copyleft (Share Alike)

Modifications to Siyarix must be distributed under the same AGPL-3.0-or-later license. This ensures that improvements benefit the entire community rather than being locked away in proprietary forks.

### Network Transparency (The "Affero" Clause -- Section 13)

The AGPL's defining feature: if you modify Siyarix and run it as a network service (SaaS, API, cloud platform) that users interact with over a network, **you must provide the complete corresponding source code** to those users. This closes the "SaaS loophole" present in the standard GPL.

### "or later" Upgrade Path

The license permits use under this version or any later version of the AGPL published by the Free Software Foundation. This simplifies future license upgrades without requiring re-licensing from every copyright holder.

### Patent Grant

Contributors to Siyarix automatically grant a non-exclusive, worldwide, royalty-free patent license for their contributions. This protects users from patent litigation related to contributed code.

### No Warranty

The software is provided "as is," without warranty of any kind, as stated in Section 11 and 12 of the AGPL.
